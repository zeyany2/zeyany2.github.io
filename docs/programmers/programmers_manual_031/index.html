    GENERAL COMMENTS ANDINDEXES 4.1 GENERAL COMMENTS ANDINDEXES TheI_ASTRAN modules (amodule isalogical group ofsubroutines) documented inthis section havebeenclassified into7categories: l)Executive Preface modules, 2)Executive modules, 3)Executive DMAPinstructions, 4)Executive DMAPmodules, 5)functional modules, 6)output modules, and7)matrix modules. Executive Preface modules arethose which areexecuted prior totheexecution ofthe first module inaD_Psequence.
